# Meilenazon

### 1. Install Dependencies:
`$ pnpm install`

### 2. Run tests
`$ npx hardhat test`

### 3. Start Hardhat node
`$ npx hardhat node`

### 4. Run deployment script
In a separate terminal execute:
`$ npx hardhat run ./scripts/deploy.ts --network localhost`

### 5. Start frontend
`$ pnpm run start`
